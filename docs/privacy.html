Política de Privacidade

NotasPat v1.2.0

Última atualização: Fevereiro de 2026

Resumo: A extensão NotasPat respeita sua privacidade. Todos os dados são armazenados localmente no seu navegador e não são transmitidos para servidores externos.

Dados Coletados

Dados Armazenados Localmente

A extensão armazena as seguintes informações no seu navegador (usando chrome.storage.local):

  • Notas: Texto das notas que você cria
  • Tags: Categorias que você atribui às notas
  • Cores: Preferências de cor para cada nota
  • Templates: Modelos de texto personalizados
  • Lembretes: Data/hora de lembretes configurados
  • Tema: Preferência de tema (claro/escuro)

Dados NÃO Coletados

A extensão NÃO coleta:

  • Informações pessoais identificáveis
  • Dados de navegação
  • Histórico de browsing
  • Localização
  • Nenhuma informação é enviada para servidores externos

Armazenamento

Todos os dados são armazenados exclusivamente no seu dispositivo, usando o armazenamento local do Chrome (chrome.storage.local).

  • Os dados ficam apenas no seu navegador
  • Não são sincronizados com servidores externos
  • Não são compartilhados com terceiros
  • Você pode exportar e excluir seus dados a qualquer momento

Compartilhamento de Dados

Não compartilhamos seus dados com ninguém.

Seus dados nunca são vendidos, alugados ou compartilhados com terceiros para fins de marketing ou qualquer outro propósito.

Exportação e Exclusão

Exportar Dados

Você pode exportar todas as suas notas a qualquer momento:

  1. Clique no ícone da extensão
  2. Clique em "Exportar"
  3. Um arquivo JSON com seus dados será baixado

Excluir Dados

Para excluir todas as suas notas:

  1. Clique no ícone da extensão
  2. Exclua cada nota individualmente, ou
  3. Desinstale a extensão (isso remove todos os dados)

Segurança

  • Os dados são armazenados localmente no seu navegador
  • Recomendamos fazer backup periódico usando a função "Exportar"
  • Notas podem conter informações sensíveis - mantenha seu arquivo de exportação em local seguro
  • A extensão utiliza práticas de segurança para prevenir ataques XSS (Cross-Site Scripting)

Atenção: Notas podem conter informações sensíveis. Ao exportar, tenha cuidado com o destino do arquivo JSON e mantenha-o em local seguro.

Crianças

Esta extensão não é destinada a crianças menores de 13 anos e não coleta informações intencionalmente de crianças.

Alterações nesta Política

Podemos atualizar esta política de privacidade ocasionalmente. Alterações significativas serão notificadas através da extensão.
